Description

Given a positive integer n and n positive integers a_1, a_2, ..., a_n. Compute the number of positive integer solutions to the equation:
                          a_1.X_1 + a_2.X_2 + . . . + a_n.X_n = M

Input
Dòng 1: n và M
Dòng 2: a_1, a_2, ..., a_n

Output
   Số nghiệm nguyên dương

Ví dụ
Input
3 5
1 1 1

Output
6
